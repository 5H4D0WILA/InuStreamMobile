# Inu's Stream (iOS/Android)

This is the mobile version of my [Inu's Stream](https://inu.watch) website
It is a simple UI design without any ads to watch anime uninterrupted.

## Features:
- [x] Search Page
- [x] Info Page
- [x] Watch Page
- [ ] Recent Episodes
- [ ] Continue Watching
- [ ] Watch List using anilist login
